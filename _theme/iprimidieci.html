package com.test;
 
public class Sort {
    // 对R[low..high]快速排序
    public static void QuickSort(Comparable[] data, int low, int high) { 
        int pivotpos; // 划分后的基准记录的位置
 
        if (low < high) {// 仅当区间长度大于1时才须排序
            pivotpos = Partition(data, low, high); // 对Comparable[] data做划分
            QuickSort(data, low, pivotpos - 1); // 对左区间递归排序
            QuickSort(data, pivotpos + 1, high); // 对右区间递归排序
        }
    } // QuickSort
 
    private static int Partition(Comparable[] data, int i, int j) {
        // 从数组两端交替地向中间扫描
        Comparable pivotKey = data[i];
        // 进行扫描的指针i,j;i从左边开始,j从右边开始
        while (i < j) {
            while (i < j && data[j].compareTo(pivotKey) > 0) {
                j--;
            }// end while
            if (i < j) {
                // 比枢纽元素小的移动到左边
                data[i] = data[j];
                i++;
            }// end if
            while (i < j && data[i].compareTo(pivotKey) < 0) {
                i++;
            }// end while
            if (i < j) {
                // 比枢纽元素大的移动到右边
                data[j] = data[i];
                j--;
            }// end if
        }// end while
        // 枢纽元素移动到正确位置
        data[i] = pivotKey;
        return i;
    }
 
    public static void main(String[] args) {
        Comparable[] c = {4, 9, 23, 1, 45, 27, 5, 2 };
        QuickSort(c, 0, c.length - 1);
        for (Comparable data : c) {
            System.out.println(data);
        }
    }
 
}

L'idea nasce un po' per caso un po' per amore.
Signorperello ha sempre avuto la mania di classificare e mettere in giudizio qualsiasi cosa. Dai pianeti (come farà poi a conoscerli tutti e giudicarli?) ai pizzoli, mettendoli in ordine dal primo al decimo.
Era una notte buia e tempestosa quando propose a due ben più colti personaggi, di visionare l'ambiziosa classifica dei 200 film più belli della storia. Fu la goccia che fece traballare il gabinetto.

Cr*st*na ed Antonio dissero: - No, questo è troppo, prima era di meno! -
(frase in classifica tra le più decisive della storia).
Stufi di sentire catalogata ogni emerita fesseria, proposero al losco individuo di classificare il mondo intero.

La sfida che lancia il nostro blog è questa: con l'aiuto delle persone, ogni singola cosa, casa, chiesa, frase, evento, animale, mollusco, obiettore di coscienza, logo di automobile, indirizzo dei pediatri di Viareggio… tutto ma proprio tutto alla fine sarà sottoposto al giudizio di Antonio, Cr*st*na e del Signorperello. Ogni singola classifica stimolerà nuove spedizioni archeologiche, stimolerà la diuresi, darà speranza agli elettrauti del Gibuti. E che nessuno si offenda se il proprio telefonino o la propria città saranno relegate al decimo posto. È già un onore far parte di una delle classifiche de I primi dieci, è questa è al primo posto delle classifiche delle minchiate di tutti i tempi.
